Svevo e l'inetto - PWA didattica

Pubblicazione locale:
1. estrai lo ZIP
2. entra nella cartella
3. avvia un server: python3 -m http.server 8000
4. apri http://localhost:8000

GitHub Pages:
- carica tutti i file nella root del repository
- attiva Pages sulla branch principale
- attendi la pubblicazione
- su iPad/iPhone usa Safari > Condividi > Aggiungi a Home
- su Android Chrome usa Installa app

Struttura:
- index.html: copertina iniziale
- una pagina per ogni lezione
- pulsante "Lezione completa" in ogni pagina
- manifest.json
- service-worker.js
- offline.html
